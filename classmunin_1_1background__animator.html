Detailed Description

A class that executes the scheduled redraw requests on an execution strand.

The expected usage of this animator is that all operations on the animated component (and probably all components for a given user interface also take place on the strand so that there are no race conditions between animation and normal operation. This requires co-ordination from a top- level user interface component, which would be the owner of both the strand and the animator.
