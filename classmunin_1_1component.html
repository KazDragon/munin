Detailed Description

An object capable of being drawn on a canvas.

A component is a fundamental piece of the munin architecture and is the base class for anything that is capable of being drawn in some way.

◆ get_preferred_size()

terminalpp::extent munin::component::get_preferred_size ( ) const

Retrieves the preferred size of this component.

The preferred size of a component is the size at which all elements can be displayed with clarity. For example, the preferred size of a static text control with the following text:
This is a
Static Text Box
would be (15, 2).
